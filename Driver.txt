웹드라이버 -> 인터페이스

웹드라이버는 인터페이스고 인터페이스는 (just providing method without bdoy)
implement WebDriver 는 크롬드라이버가 웹드라이버 method 사용에 동의했다는 뜻이다.
마찬가지로 파이어폭스, 사파리 등도 웹드라이버 인터페이스사용에 동의하고 method 를 사용할수 있음.

---------------------------------------------

셀레니엄이 chromedriver.exe 파일 (중개인같은 역할) 을 invoke 할거임.
그리고 chromedriver.exe 파일이 chrome 브라우저를 invoke 할거임

chrome driver.exe 파일을 invoke 하는 방법에는 2가지가 있음.

1. Set Manually
System.setProperty("webdriver.chrome.driver", "path of .exe file");

2. Selenium Manager
WebDriver driver = new ChromeDriver();
